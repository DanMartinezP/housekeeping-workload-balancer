# Housekeeping Workload Balancer

An automation tool designed to solve real-world operational challenges in resort housekeeping.

## Key Features
- **Weighted Assignment:** Accounts for different employee productivity levels (1.2x, 1.4x).
- **Geographical Optimization:** Minimizes walking time between resort Stages.
- **Priority Logic:** Automatically prioritizes Arrivals over Departures.
- **Cascading Workload:** Soft-constraint system for balanced distribution in high-occupancy scenarios.

## Technical Stack
- Python 3.x
- Data structures: Dictionaries, Lists, Tuples.
